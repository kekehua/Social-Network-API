# Social Network API

## Description
A social network api used to handle large amounts of data

## Technology
Key technologies and npm packages used in this project:
-JS
-Mongodb
-Express.js
-Mongoose ODM
-Moment.js

## Installation
clone repo files
have mongodb installed
run npm i
run node server
open insomnia and use api routes
